# Ramanjeet Singh - Personal Portfolio

This is the repository for my personal developer portfolio website, hosted live via GitHub Pages.

🔗 **Live Website**: [https://ramanjeetme-a11y.github.io](https://ramanjeetme-a11y.github.io)

---

## 🚀 Features

- **Responsive Layout**: Designed with a mobile-first approach, displaying beautifully on mobile, tablet, and desktop screens.
- **Ambient Visuals**: Clean dark-mode theme utilizing modern color palettes (neon accents) and smooth background glow elements.
- **Typewriter Effect**: An interactive roles display showcasing core engineering focuses.
- **Direct Resume Download**: Downloadable access to my latest SDE-2 resume.
- **Contact Form**: An interactive form for direct inquiries.

---

## 🛠️ Technology Stack

- **HTML5**: Semantic markup for layout structure.
- **Vanilla CSS**: Custom styling, grid and flexbox layouts, keyframe animations, and custom theme variables.
- **Vanilla JavaScript**: Pure JS for mobile navigation toggles, typing animation logic, scroll-based animations, and form submission simulation.
